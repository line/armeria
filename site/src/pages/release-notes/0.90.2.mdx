## Security

This release updates Netty from 4.1.38 to 4.1.39 to address the HTTP/2 security issues found in its previous versions. Please upgrade as soon as possible if your application serves the traffic from untrusted environment such as the Internet. See [Netty 4.1.39 release news](https://netty.io/news/2019/08/13/4-1-39-Final.html) and [Netflix security bulletins](https://github.com/Netflix/security-bulletins/blob/master/advisories/third-party/2019-002.md) for more information.

## Dependencies

- Brave 5.6.9 -> 5.6.10
- gRPC 1.22.1 -> 1.22.2
- Netty 4.1.38 -> 4.1.39
- SLF4J 1.7.27 -> 1.7.28
