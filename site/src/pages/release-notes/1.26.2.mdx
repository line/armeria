## 🛠️ Bug fixes

- The same `EventLoop` is used for an <type://Endpoint> within an <type://EndpointGroup> when
  used in conjunction with a <type://RetryingClient>. This ensures efficient connection reuse. #5290
- You no longer see an <type://EndpointSelectionTimeoutException> when selecting an <type://Endpoint> from an
  <type://EndpointGroup>. #5281
- Fixed a bug where Armeria Spring integration is not compatible with Spring Boot versions 2.6 and earlier. #5279
