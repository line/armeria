## 🛠️ Bug fixes

- <type://Server> now aborts the startup process if <type://ServerListener#serverStarting(Server)>
  raises an exception. #3935 #3947
- You no longer see an <type://EmptyEndpointGroupException> when the entire <typeplural://Endpoint>
  are replaced in a <type://HealthCheckedEndpointGroup>. #3927
- You no longer see an `IllegalStateException` in your gRPC calls when an <type://EndpointGroup> is
  empty. #3951

## 🔒 Security fixes
