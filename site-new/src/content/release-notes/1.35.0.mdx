- **Per-Request Client TLS Configuration**: You can now specify [ClientTlsSpec](type) for each request using
  [RequestOptions#clientTlsSpec()](type) or [ClientRequestContext#clientTlsSpec()](type). #6551
- **Nested Context Paths**: You can now use nested context paths to organize your service routes more effectively. #5758 #5846
  ```java
  Server.builder()
        .contextPath("/rest")
            .contextPath("/catalog")  // ğŸ‘ˆğŸ‘ˆğŸ‘ˆ

                .and()

                          .and()
        .contextPath("/gql")
            .service("/catalog", new GraphQLService()); // /gql/catalog
  ```
- **WebSocket Stream Timeout**: You can now set `streamTimeout` option to [WebSocketService](type) and [WebSocketClient](type). #6357
```java

                .streamTimeout(Duration.ofMinutes(1)) // ğŸ‘ˆğŸ‘ˆğŸ‘ˆ

               .build();
  ```
- **Athenz Spring Integration Enhancements**: You can now customize metrics configuration. #6566
  - You can customize [ZtsBaseClient](type) via `Consumer<ZtsBaseClientBuilder>`.
- **RequestLogListener**: You can now use [RequestLogListener](type) to observe all [RequestLog](type) events. #6543
- **Connection Pool Ping Events**: [ConnectionPoolListener](type) now supports ping-related events. #6539 #6552
- **DocService UI Customization**: You can now customize the appearance of DocService, including the title, header background color, and button colors. #6235
