# Control planes

Armeria can interoperate with control planes that provide the supported Envoy v3
resources and gRPC transport variants. See
[Supported features](/docs/advanced/xds/supported-features) for the full list of
recognized fields.

- [**java-control-plane**](https://github.com/envoyproxy/java-control-plane) — a Java
  library for building xDS control planes. Armeria's xDS module is extensively tested
  against java-control-plane.
- [**go-control-plane**](https://github.com/envoyproxy/go-control-plane) — a Go library
  for building xDS control planes.
