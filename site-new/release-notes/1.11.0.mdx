# v1.11.0

## 🌟 New features

- You can now easily check if the system has a working IPv6 stack using [SystemInfo#hasIpV6()](type). #3784

{/* truncate */}

## 📈 Improvements

- The cost of creating an [Endpoint](type) is reduced by cache. #3767 #3772

## 🛠️ Bug fixes

- No more leaks when using [HttpResponse#from(CompletableFuture)](type). #3796
- Armeria Thrift client now wraps a `RuntimeException` with `TTransportException` except
  [UnprocessedRequestException](type) and [FailFastException](type). #3794
- The value of [RequestHeaders#path()](type) is now properly encoded. #3780 #3781
- You can now access [ServiceRequestContext](type) via [ServiceRequestContext#current()](type)
  in a GraphQL service.

## ⛓ Dependencies

- gRPC 1.40.0 → 1.40.1
- Reactor gRPC 1.1.0 → 1.2.0
- Spring Boot 2.5.3 → 2.5.4
